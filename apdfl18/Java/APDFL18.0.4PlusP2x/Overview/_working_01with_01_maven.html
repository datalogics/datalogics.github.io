Working with Maven

APDFL can be used with Maven. You can obtain the license managed SDK from Maven Central, or deploy the non-license managed SDK to an internal Maven artifact server.

Using Maven with the license managed SDK

A license managed copy of the SDK can be obtained through Maven Central.

Using Maven with the non-license managed SDK

If developing on Windows, see Installing Maven artifacts on Windows.

If developing on Other Platforms, see Installing Maven artifacts on Other Plaforms.

Adding the SDK to a Maven project

Once the SDK is available in a Maven repo, you can add it to a Maven project, see Adding the SDK to a Maven project.
