Detailed Description

A curveto operator where the first control point matches the beginning.

The current point is the beginning point in the curve.

The name CurveToV is drawn from the "V" path construction operator. See "Path Construction and Painting," section 8.5.2.1 of the ISO 32000-1:2008, Document Management-Portable Document Format-Part 1: PDF 1.7, page 132.
